GNU/Linux is a open source operating system, it consists of Linux kernel and a piece of programs. It has many distributions, RHEL/CentOS、 Debian/Ubuntu、SUSE/OpenSUSE are mainstream distributions. In order to extract the product support life cycle info, I wrote a Shell script to extract relevant info from their official site.

GNU/Linux has 3 major mainstream distributions: RHEL、Debian、SUSE。RHEL、SUSE are maintenanceed by commercial corporation, Debian is maintenanceed by community, but Ubuntu (the derivative of Debian) is maintenanceed by commercial corporation.

EOL Status

My system info check script gnuLinuxMachineInfoDetection.sh uses these data to determine if the distro release version is out of maintenance.

Ubuntu Release EOL

Ubuntu has listed release date and EOL date of its release version (include unreleased) in official page Ubuntu release end of life. But the data has been hidden using HTML class class="u-hide--medium u-hide--large", it just be shown on mobile device.

curl -fsL https://www.ubuntu.com/info/release-end-of-life | sed -r -n '/Ubuntu Server and desktop release end of life/,/<\/tbody>/{/<tbody>/,${s@^[[:space:]]*@@g;s@<\/tr>@---@g;p}}' | sed ':a;N;$!ba;s@\n@@g;s@---@\n@g;' | sed -r -n 's@<\/td>@|@g;s@<[^>]*>@@g;s@\|$@@g;s@&nbsp;@@g;/^$/d;p' | awk -F\| 'BEGIN{OFS="|"; print "Release|Release Data|End of Life\n---|---|---"}{print}'
